Bootstrap

Es una libreria o framework que nos facilita el uso de dar estilos de css de manera resposiva, todo esto elaborado por un sistema de grillas que esta conformado de 12 columnas, ademas que para agregar un diseño solo se deben usar las clases ya definidas, pero antes de usar esto, te recomiendo entender el fundamento basico o intermedio de CSS y Javascript.

Desarrollador, Josue Huayapa

Mobile First

En el transcurso de los años cada vez se puede ver a una persona con algun dispostivo moviles, eso hace que el la web se deba dar mucha prioridad a estos dispotivos, es por eso que se creo la estrategia llamada "Mobile First", ya que su desarrollo web se basara primeramente en los dispositivos moviles para que su diseño se adapte a pantallas mas grandes como un ordenador o un smart TV
